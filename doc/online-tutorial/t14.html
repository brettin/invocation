Now that our model has been gapfilled, it should be possible to simulate biomass production using flux balance analysis. Let's try the runfba command again: 
  runfba < gapfilled.model > solution.html

 

Once this command completes, run an ls and click on the html printed by the FBA command. Your FBA solution should now include a nonzero objective value as well as numerous compound and reaction fluxes.
