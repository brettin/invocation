v0.0.5 - 11/18/2013

  • Scriptability - save your commands to a file (one command per line), then run it later with execute nameofscript
  • Multiple commands can be run at once - delimited commands with semicolons. echo a ; echo b
  • Moved the interface to bootstrap v3.0

v0.0.4 - 09/23/2013

  • Minor bug fixes
    • Files can now be downloaded
    • Variables can now be re-assigned
    • Variables are cleared from session upon logout
    • Error messages no longer auto clean up

v0.0.3 - 08/23/2013

  • User defined variables -
    $genome = kb|g.1994
    all_entities_Genome | grep $genome
  • Help for commands - click on the ? next to the command name in the list to the left for usage info.
  • Iris now supports uploading huge files (up to at least multiple gigabytes), and uploads are resumable if there's a failure.
    Likewise, Iris now supports real file downloads to get your processed information back out of the system.
    If you have a stalled upload job, just mouseover it in the process list to resume it or cancel the upload.

    Finally, you can also upload via the command line - upload filename
  • Enhanced file browser. Mouseover the items to interact with them.
    • Add a directory.
    • Upload a file.
    • Delete a file or directory.
    • Download this file.
    • Edit this file.
    • Add this file to your terminal input.
  • Iris has a simple file editor - in the file browser, mouse over a file name and click the pencil to tweak (small) files.
  • Images are viewable. Type view filename
  • Lots of behind the scenes infrastructure improvements.
  • Iris supports natural language questions. Type questions to bring up a list of natural language prompts.
    You can also type them on the command line and hit arrow-right to attempt an auto completion. Further, you can arrow-right to select the next variable in the sequence.
  • Improved command search functionality. Just start typing into the search box under the list of commands to filter down to what you want to see.
  • Add comments to your output by typing #your comment here
  • Terminate processes by mousing over the process list and clicking the cancel button.
